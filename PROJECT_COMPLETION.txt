================================================================================
                            📞 快速开始
================================================================================

1. 安装依赖
   cd backend && npm install
   cd ../frontend && npm install

2. 启动后端（终端1）
   cd backend && npm start

3. 启动前端（终端2）
   cd frontend && npm run dev

4. 访问应用
   前端: http://localhost:3001
   后端: http://localhost:3000
   默认账号: admin / admin123
